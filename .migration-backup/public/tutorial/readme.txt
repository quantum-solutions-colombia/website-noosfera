# Imágenes del Tutorial de Noösfera

Este directorio contiene las imágenes utilizadas en el tutorial interactivo de Noösfera.

- welcome-dashboard.jpg: Pantalla de bienvenida al sistema Noösfera
- bci-connection.jpg: Ilustración de la conexión del dispositivo BCI
- thought-capture.jpg: Captura de patrones neuronales
- neural-visualization.jpg: Visualización 3D de patrones neuronales
- content-generation.jpg: Interfaz de generación de contenido
- brainwave-analysis.jpg: Análisis de ondas cerebrales
- content-library.jpg: Biblioteca de contenido generado
- system-settings.jpg: Panel de configuración del sistema
- ready-to-start.jpg: Pantalla final del tutorial
