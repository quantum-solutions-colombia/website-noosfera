# Noösfera system

Sistema avanzado de captura y análisis de patrones cardíacos con generación de contenido basado en datos fisiológicos.

## 🚀 Características

- **Captura de datos cardíacos en tiempo real** - Monitoreo continuo de ritmo cardíaco y variabilidad
- **Generación de contenido inteligente** - Análisis de texto e imágenes basados en datos fisiológicos
- **Dashboard administrativo** - Panel completo con métricas y estadísticas en tiempo real
- **Almacenamiento local** - Todos los datos se guardan localmente en el navegador (localStorage)
- **Modo demo** - Prueba el sistema sin necesidad de dispositivos reales

## 📦 Tecnologías

- **Next.js 14** - Framework de React con App Router
- **TypeScript** - Tipado estático para mayor seguridad
- **Tailwind CSS** - Estilos utilitarios y diseño responsive
- **Framer Motion** - Animaciones fluidas y transiciones
- **Recharts** - Visualización de datos y gráficos
- **localStorage** - Almacenamiento de datos en el navegador
- **Vercel Blob** - Almacenamiento de imágenes generadas

## 🛠️ Instalación

\`\`\`bash
# Instalar dependencias
npm install
# o

# Iniciar servidor de desarrollo
npm run dev
# o

\`\`\`

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 💾 Sistema de Almacenamiento

El sistema utiliza **localStorage** para almacenar:
- Usuarios y credenciales
- Sesiones BCI
- Pulsos cerebrales capturados
- Imágenes generadas
- Configuración del usuario

**Nota**: Los datos se almacenan localmente en tu navegador. Si limpias los datos del navegador, se perderá toda la información.

### Dashboard de Usuario
- Visualización de patrones cardíacos en tiempo real
